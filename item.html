Добриво Compo для кактуса та бонсаю

Артикул: 000259
Замовити
  • Опис
  • Характеристики
  • Відгуки (7)

Рідке добриво для кактуса й бонсаї від Compo. Формула NPK: 5-5-7 з мікроелементами та низьким рівнем хлориду. Екологічно безпечне.

Компанія Сompo — виробник мінеральних добрив і торфосуміші для домашнього квітникарства, садівництва, професійного ландшафтного озеленення. Компанія на ринку добрив уже понад 60 років, з 1956 р.

Сompo — лідер європейського ринку в сегменті біологічних, хімічних продуктів для дому та саду, спеціалізованих добрив для професійного використання. Сьогодні добрива та торфосуміші Compo експортуються у понад 100 країн світу.

Особливості Compo для кактуса й бонсаї:

  1. Ідеальне поєднання поживних речовин для всіх кактусів, сукулентів, а також бонсаї.
  2. Має миттєвий ефект.
  3. У складі є додатковий калій для швидкого росту.
  4. Містить усі необхідні мікро- та макроелементи.
  5. Підвищує імунітет рослин.

Формула NPK:

  • N (азот): 5%
  • P2O5 (фосфор): 5%
  • K2O (калій): 7%

Країна-виробник добрива: Німеччина, тому у виробництві враховано всі екологічні та оборонні норми згідно зі стандартами ЄС.

  • 1 раз на тиждень/7 мл добрива на 1 л води
  • Період внесення — березень-жовтень.
  • Призначення Для бонсаїв
  • Тип Комплексні
  • Склад Азотні, Калійні, Фосфорні
  • Вид Рідкі
  • Підходить Для кактусів
  • Фасування Міні-упаковка (до 1 л/кг)
  • Упаковка Пляшка
  • Спосіб внесення Кореневе
  • Період внесення Весна
  • За складом NPK 5 + 5 + 7 з МО
  • Країна-виробник товару Німеччина
  • Гарантія 14 днів
